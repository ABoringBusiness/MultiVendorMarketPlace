# Multi-Vendor Bidding System

The Multi-Vendor Bidding System is a platform designed to connect buyers and sellers through a dynamic auction process. Sellers can post products for auction, and buyers can place bids. The highest bidder wins the product when the auction ends. This repository contains the source code and documentation for the project.

## Features

- **User Authentication**: Secure login and registration for users.
- **Bidding System**: Participate in auctions with real-time updates
- **User Profiles**: Create and manage detailed user profiles.
- **Auction Listings**: Browse and search for active auctions.
- **Email Notifications**: Receive updates for auctions and bids.
- **Email Notifications**: Receive updates for auctions and bids.
- **Admin Dashboard**: Manage users, auctions, and reported content.
- **Reports & Moderation:**: Report auctions and allow admin review.
- **Category Suggestions**: Suggest new categories for the platform and allow admin review.
- **Responsive Design**: Optimized for both desktop and mobile platforms.

## Technologies Used

- **Frontend**: React.js, Redux, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Email Templating**: EJS
